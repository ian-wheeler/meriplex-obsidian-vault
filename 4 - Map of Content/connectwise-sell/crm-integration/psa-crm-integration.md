# ConnectWise PSA CRM Integration Overview
---
The following collection of documentation can help you set up and configure your ConnectWise PSA™ (Manage) integration with ConnectWise CPQ™ (Sell). This integration enables you to pull opportunities, products, contacts, and account information into Sell and pass quote information back to PSA.
