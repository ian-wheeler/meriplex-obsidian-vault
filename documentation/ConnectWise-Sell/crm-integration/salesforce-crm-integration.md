# Salesforce CRM Integration Overview
---
The following collection of documentation can help you set up and configure your Salesforce® integration with ConnectWise CPQ™ (Sell). This integration enables you to pull products, contacts, and account information into Sell and pass quote information back to Salesforce. Refer to the documentation below to troubleshoot your Salesforce integration, authorize users, and edit Salesforce settings in CPQ.
