# Company Overview

Company enables you to manage your internal company settings and enable ConnectWise single sign-on (SSO) to make the ConnectWise CPQ™ (Sell) login process simpler. For more information on how to set specific office locations, refer to Locations documentation.  

Tip: To locate this screen navigate to Settings > Organization Setup > Company. ## Field Descriptions — [[cwsell-company-matrix.xlsx]] ## SSO Setup — After you have set up an administrator account with the ConnectWise Home, you can set up your ConnectWise SSO connection with ConnectWise CPQ (Sell). Any CPQ usernames that do not have an SSO account are submitted to the ConnectWise Home and an email is sent to them so they can sign up. Refer to SSO Setup for CPQ for instructions on how to set up ConnectWise SSO. ## How to Update Company Logo — Company logos display on the Order Porter page and quote PDF. Refer to Logo and Image Specifications for file specifications. 

  1. Navigate to Settings > Organization Setup > Company > then click anywhere in the Company Logo field.
  2. Choose one of the following options from the Logo window:
    1. Choose File: Click to import file of an image saved locally. 
      • Fix to crop square: Select the check box to automatically crop the logo field into a square.
      • Delete: Delete the current logo.
      • Cancel: Cancel all current edits and changes.
      • Save: Save all changes.
      • Help: Provides overlay with field descriptions of the Logo window.
    2. Input an image URL: Enter a public URL of an image.
      • Fix to crop square: Select the check box to automatically crop the logo field into a square.
      • Delete: Delete the current logo.
      • Cancel: Cancel all current edits and changes.
      • Save: Save all changes.
      • Help: Provides overlay with field descriptions of the Logo window.
  3. Click Save.
